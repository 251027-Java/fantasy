Fantasy is an analytical fantasy football web application based on [Sleeper](https://sleeper.com/). It features custom statistics showcasing your performance in your leagues.

Fantasy is not affiliated with [Sleeper](https://sleeper.com/) or any other organization shown above.

## Development

Clone the repository:

```sh
git clone https://github.com/251027-Java/fantasy.git
cd fantasy
```

Further documentation can be found on how to set up the [frontend](frontend/README.md) and [backend](backend/README.md) in their respective directories' README.
